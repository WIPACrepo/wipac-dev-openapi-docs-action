# wipac-dev-openapi-docs-action
GitHub Action Package for Auto-Generating Documentation from a Project's OpenAPI Schema File

## _Deprecated as of March 2026_

This action package has been deprecated. Instead, projects will use Sphinx tooling, as seen in https://github.com/Observation-Management-Service/ewms-docs.

This repository previously hosted the WIPAC Dev Team's GitHub Action package to automate documentation for OpenAPI specs.
